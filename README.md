
# Nanocell - CSV

A csv file viewer and editor.

- fast
- simple
- lightweight
- open-source
- cross platform
- quick-view large files
- is a PWA (Progressive Web App)
- works 100% offline
- custommizable
- free
  

App download availabale at : [https://nanocell-csv.com](https://nanocell-csv.com)
